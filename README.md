# Job Application Tracker

A Python command-line application for tracking job applications and application status.

## Features

- Add new job applications
- View all applications
- Update application status
- Delete applications
- View application summary
- Store data locally

## Technologies Used

- Python
- CSV File Handling

## How to Run

1. Clone the repository

```bash
git clone https://github.com/olokunde/job-application-tracker.git
```

2. Navigate to the project folder

```bash
cd job-application-tracker
```

3. Run the application

```bash
python job_tracker.py
```

## Example Use Cases

- Track applications sent on Upwork
- Monitor internship applications
- Organize remote job applications
- Keep record of application statuses
